TAGs i, u, strong

A tag "i" é usada para deixar uma palavra ou as palavras selecionadas em itálico. EX: Eu sou o milior

A tag "u" é usada para sublinhar um determinado texto selecionado. EX: Eu sou o milior!

A tag strong é usada para deixar um texto selecionado em negrito. EX: Eu sou o milior!!

OBS essas tags podem ser combinadas, ou seja o texto pode ser ao mesmo tempo sublinhado, italico e negrito, para isso é necessário encadear as TAGS. EX: EU SOU O MILIOR!!!!!

Tags "font, del, abbr"

A tag font é usada para formatar uma fonte. OBS: não é recomendado o uso dela em HTML5. ex: texto editado


A tag del indica que um texto foi retirado de um texto, e pode indicar correção de um texto ex: Eu sou o milior


A tag abbr é usada para definir uma abreviação ou um acrônimo em um texto. ex: ONU
